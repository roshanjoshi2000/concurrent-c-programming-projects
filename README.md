# c-programming-concurrency-projects
C programming assessment covering linear regression, pthread-based Pi calculation, polynomial derivatives, and Sobel edge detection using lodepng.

## Projects Included

### 1. Linear Regression
- Implemented linear regression in C
- Processed numerical data from input files
- Calculated prediction values using statistical formulas

### 2. Pi Approximation with Pthreads
- Used POSIX threads to estimate the value of Pi
- Split computation across multiple threads
- Improved performance using concurrent execution

### 3. Polynomial Derivatives with Pthreads
- Parsed polynomial expressions from input files
- Used dynamic memory allocation with `malloc`
- Applied pthreads to process derivative calculations concurrently

### 4. Sobel Edge Detection
- Implemented Sobel edge detection in C
- Used `lodepng` for PNG image processing
- Applied pthreads to improve image processing performance
